Endpoint Usage Guide

Welcome to the Endpoint Usage Guide. Here's how you can interact with the endpoints:

GET Request:

To retrieve information from an existing endpoint, use a GET request. If the endpoint exists, you will receive the resource content.

PUT Request:

To modify an existing resource, use a PUT request to the endpoint. Provide the updated data in the request body to make changes.

DELETE Request:

To delete an existing resource, use a DELETE request to the endpoint. This will remove the resource from the server.

POST Request:

To create a new resource, use a POST request to a non-existing endpoint. If the endpoint does not exist, the server will add the new resource.

Remember to include the necessary headers and data in your requests to interact with the endpoints effectively.
